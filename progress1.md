# Status Report 1 — Forsaken

## Reporting Window

Early implementation milestone focused on building the playable foundation.

## Objectives for This Milestone

- Build core player movement (walk/jump/crouch).
- Stand up baseline UI (pause/menu behavior).
- Create the technical foundation for combat (scriptable weapon objects).
- Establish team delivery rhythm and task ownership.

## Progress Summary

Completed work:

- Implemented movement mechanics including jumping and crouching.
- Added functional pause menu UI and pause audio cue.
- Created scriptable weapon objects for future combat integration.
- Finalized initial boss concepts tied to grief-themed progression.
- Started music and sound design collaboration for menu/gameplay tracks.

## Risks and Gaps

- Weapon data systems are in place, but gameplay hookup remained pending.
- Formal external feedback had not yet been received at this stage.

## Team Workflow

- Weekly sync meetings with daily chat coordination.
- Task tracking through GitHub Projects.
- Shared visibility on in-progress vs. complete features.

## Next Milestone Priorities

- Implement enemy movement (patrol and chase behavior).
- Integrate initial melee combat functionality.
- Add and test early audio integration.
- Use boss concepts to guide level and narrative scaffolding.
- Begin design for boss resurrection/control-transfer mechanics.
